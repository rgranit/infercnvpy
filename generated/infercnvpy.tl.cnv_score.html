infercnvpy.tl.cnv_score¶

infercnvpy.tl.cnv_score(adata, *, obs_key='cnv_leiden', use_rep='cnv', key_added='cnv_score', inplace=True)¶

Assign each cnv cluster a CNV score.

Clusters with a high score are likely affected by copy number abberations. Based on this score, cells can be divided into tumor/normal cells.

Ths score is currently simply defined as the mean of result of infercnvpy.tl.infercnv() for each cluster.

Parameters
adata : AnnDataAnnData

annotated data matrix

obs_key : strstr (default: 'cnv_leiden')

Key under which the clustering is stored in adata.obs. Usually the result of infercnvpy.tl.leiden(), but could also be other clusters, e.g. obtained from transcriptomics data.

use_rep : strstr (default: 'cnv')

Key under which the result of infercnvpy.tl.infercnv() is stored in adata.

key_added : strstr (default: 'cnv_score')

Key under which the score will be stored in adata.obs.

inplace : boolbool (default: True)

If True, store the result in adata, otherwise return it.

Returns

Depending on the value of inplace, either returns None or a vector with scores.
